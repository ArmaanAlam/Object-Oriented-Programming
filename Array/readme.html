Object Concepts in C++

Visualizing the difference between an object containing arrays and an array of objects.

Concept 1: Object Containing Arrays

A single object manages internal arrays. Example: `Shop dukaan;`

Shop Object (`dukaan`)

int counter;

3

int itemId[100];

101
102
103
...

int itemPrice[100];

150
200
50
...

Concept 2: Array of Objects

An array where each element is a separate, independent object. Example: `employee harry[5];`

`harry` Array

harry[0]

id: 1

salary: 10000

harry[1]

id: 2

salary: 10000

harry[2]

id: 3

salary: 10000

... and so on
